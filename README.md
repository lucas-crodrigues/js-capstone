## Getting Started

To clone the repo run the line `git clone https://github.com/lucas-crodrigues/js-capstone.git` on your console.

To get a local copy up and running, run these command lines on your console:<br>
  1- `cd js-capstone` <br>
  2-  `npm install` <br>
  3-  `npm start` <br>
